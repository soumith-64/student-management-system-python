# 📄 Student Record Format

Every student is stored like this:

```
RollNo|Name|Age|Department|Python|Math|English|Total|Average|Grade|Status
```

Example

```
101|Soumith|18|CSE|95|91|89|275|91.67|A+|Pass
```
